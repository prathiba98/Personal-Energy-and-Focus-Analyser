 #  Personal Energy & Focus Analyzer

 ## streamlit app link: https://personal-energy-and-focus-analyser-bnszspqznypvjtusb3kjbq.streamlit.app/ 

An interactive Exploratory Data Analysis (EDA) dashboard built using **Python and Streamlit** to analyze daily patterns in energy, focus, mood, sleep, and screen time.

The project uses a **synthetically generated but realistic behavioral dataset**, enabling complete control over data quality, assumptions, and analysis logic.

---

##  Project Overview

This project explores how lifestyle factors such as sleep duration and screen time influence:
- Focus levels
- Energy levels
- Overall productivity

The analysis is presented through an interactive Streamlit dashboard designed for intuitive exploration and storytelling.

---

##  Key Features

-  KPI overview for focus, energy, and productivity  
-  Workday vs Weekend comparison  
-  Sleep vs Focus analysis  
-  Screen Time vs Energy analysis  
-  Weekly productivity trends  
-  Correlation analysis across behavioral factors  

---

## Tech Stack

- Python  
- Pandas  
- Matplotlib  
- Streamlit  

---

##  Dataset

- Generated using Python with realistic constraints
- Covers **1 year of daily data**
- Includes:
  - Sleep hours
  - Screen time
  - Mood score
  - Energy level
  - Focus level
  - Workday/Weekend indicator

> No external datasets (e.g., Kaggle) were used.
